# TransitMind AI 业务用户门户 - 前端项目

> Agent-Native 交通政务智能体操作系统前端实现

## 项目结构

## 核心功能模块

### 1. 智能交互中枢（Dashboard）
- 主动式智能体推送（Proactive Agent）
- 自然语言输入框
- 我的智能体矩阵
- 待办事项卡片

### 2. 多智能体协同可视化（AgentWorkflow）
- 5种智能体类型：协调/分析/优化/验证/执行
- 动态节点发光效果
- 数据流动连线动画
- 实时工作流控制

### 3. 沉浸式智能体空间（AgentSpace）
- 三栏布局：控制栏 + 对话/编辑区 + 工具区
- 多轮对话交互
- 实时文档编辑
- 任务进度追踪

### 4. 智能体市场（AgentMarketplace）
- 分类浏览（规划/合规/项目/应急/数据）
- 一键订阅/取消
- 评分与评论
- 使用统计

## 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **图标**: Lucide React
- **Mock**: MSW (Mock Service Worker)

## 科技感设计特色

### 配色系统
- 主色: 深空蓝 `#002B5B` + 极光青 `#00E0FF`
- 辅助色: 智能体紫 `#7B61FF`、合规绿 `#00C48C`
- 背景: 深空灰 `#121A29` + 动态粒子效果

### 动效设计
- 节点发光呼吸效果
- 数据流动连线动画
- 毛玻璃半透明面板
- 渐变发光边框

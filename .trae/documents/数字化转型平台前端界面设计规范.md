# 数字化转型平台前端界面设计规范

> 基于 dtxp.chinadep.com 参考设计

## 一、整体风格定位

### 1.1 设计理念
- **专业稳重**：面向企业级用户，体现专业性和可信度
- **简洁高效**：减少视觉干扰，突出核心功能
- **数据驱动**：强调数据展示和分析能力
- **统一规范**：保持全局一致性

### 1.2 视觉风格
- 扁平化设计为主，适度使用阴影层次
- 卡片式布局，模块化组织内容
- 清晰的视觉层级和导航结构
- 响应式设计，适配多终端

- 使用 Figma/Sketch 进行设计
- 提供 1x、2x、3x 切图
- 标注颜色、字体、间距、圆角等属性
- 提供交互说明文档

### 22.2 开发协作
- 设计稿与代码保持同步
- 使用设计令牌 (Design Tokens) 管理变量
- 定期进行设计走查

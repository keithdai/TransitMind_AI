### 2.3 智能体详情页修改

每个智能体需要包含以下完整信息：

```typescript
interface Agent {
  id: string
  name: string
  description: string
  category: string
  icon: string
  color: string
  rating: number
  users: number
  tags: string[]
  subscribed: boolean
  features: string[]
  knowledgeBases: string[]
  recommended?: boolean
  // 新增字段
  phase: 'MVP' | 'T2'  // 开发阶段
  coreCapabilities: string[]  // 核心能力
  demoScenarios: DemoScenario[]  // 演示场景
}

interface DemoScenario {
  title: string
  userInput: string
  agentResponse: string
  highlights: string[]
}
```

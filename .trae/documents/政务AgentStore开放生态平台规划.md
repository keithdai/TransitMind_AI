### 4.3 工作流节点类型

| 节点类型 | 功能       | 说明         |
| ---- | -------- | ---------- |
| 输入节点 | 接收用户输入   | 文本/文件/语音   |
| 知识检索 | 检索知识库    | 向量/关键词/混合  |
| 模型推理 | LLM 调用   | 支持多模型切换    |
| 工具调用 | API/函数调用 | 自定义工具集成    |
| 条件分支 | 条件判断     | if-else 逻辑 |
| 循环节点 | 批量处理     | 迭代执行       |
| 变量操作 | 变量存取     | 上下文管理      |
| 输出节点 | 结果输出     | 文本/文件/图表   |

## 五、主题配色优化

### 5.1 现有主题分析

| 主题   | 当前配色        | 问题          |
| ---- | ----------- | ----------- |
| 深色主题 | 深蓝黑底 + 青色点缀 | 科技感强，但政务感不足 |
| 浅色主题 | 白底 + 蓝色点缀   | 对比度可优化      |
| 护眼主题 | 暖色调         | 可增加舒适度      |
| 政务风格 | 红色为主        | 可更精致        |

### 5.2 优化建议

#### 深色主题（科技政务）

```css
--bg-primary: #0a1628      /* 深邃蓝黑 */
--bg-secondary: #132238    /* 次级背景 */
--accent-primary: #00d4ff  /* 科技青 */
--accent-secondary: #7c3aed /* 紫色点缀 */
--text-primary: #f0f4f8    /* 主文字 */
--text-secondary: #94a3b8  /* 次级文字 */
--border: rgba(0, 212, 255, 0.2) /* 边框 */
--glow: rgba(0, 212, 255, 0.15)  /* 发光效果 */
```
